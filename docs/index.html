Caveman: menos tokens, mesma entrega

A skill caveman-terse-mode faz o Hermes cortar enrolação, manter termos técnicos exatos e responder com máxima densidade.

entrada do aluno
/skill caveman-terse-mode

Explica por que meu componente React fica renderizando toda hora e como corrigir.

Resposta comum

O motivo pelo qual o componente está renderizando novamente provavelmente está relacionado à criação de uma nova referência de objeto a cada renderização...

Resposta Caveman

Nova referência a cada render. Prop com objeto inline força re-render. Use useMemo.

cd hermes-infra-skills
mkdir -p ~/.hermes/skills
cp -a .hermes/skills/curso-hermes ~/.hermes/skills/
cp -a .hermes/skills/forumtelecom ~/.hermes/skills/
depois no Hermes
